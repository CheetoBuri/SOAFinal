# Background Images

Thư mục này chứa ảnh nền cho các trang khác nhau của website.

## Các file ảnh nền

Bạn có thể sử dụng các file sau:

1. **`cafe-landing.jpg`** - Ảnh nền cho trang landing (màn hình đầu tiên)
2. **`cafe-main.jpg`** - Ảnh nền cho trang chính (menu order)
3. **`cafe-auth.jpg`** - Ảnh nền cho trang login/register

Hoặc dùng chung 1 file cho tất cả: **`cafe-bg.jpg`**

## Hướng dẫn sử dụng

1. Thêm ảnh nền cafe của bạn vào thư mục này
2. Đặt tên file theo mục đích sử dụng (xem bên trên)
3. Kích thước khuyến nghị: 1920x1080 hoặc lớn hơn
4. Format: JPG hoặc PNG
5. Tối ưu hóa ảnh để giảm dung lượng (khuyến nghị < 500KB)

## Gợi ý ảnh nền

- **Landing**: Ảnh cafe tối màu, dramatic, có độ tương phản cao
- **Main**: Ảnh cafe sáng, ấm áp, không quá chi tiết
- **Auth**: Ảnh cafe nhẹ nhàng, tối giản
- Ảnh hạt cafe, quầy bar, không gian hiện đại
